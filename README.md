# IDM Trial Freeze Tool v1.2 Pro

## 📌 Overview
**IDM Trial Freeze Tool** is a powerful utility designed to manage and extend the trial period of Internet Download Manager (IDM). This tool provides multiple methods to handle IDM's trial limitations, including freezing the trial period, resetting the trial counter, and activating the software with custom registration details.

## ✨ Features

### 🔒 **Freeze Trial**
- Permanently freeze the 30-day trial period
- Most reliable method to keep IDM in trial mode indefinitely
- Blocks registration popups permanently
- One-click solution with automatic process handling

### 🔄 **Reset Trial**
- Reset the trial period back to 30 days
- Cleans all IDM registry entries
- Full trial counter reset
- Warning: Deletes all IDM registration keys

### ✅ **Activate IDM**
- Register IDM with custom name
- Professional activation method
- Flexible registration system
- Instant activation without restart

### 📊 **IDM Status Check**
- Real-time status monitoring
- Displays current IDM version
- Shows remaining trial days
- Visual status indicators:
  - 🟢 Green: Registered/Frozen
  - 🟠 Orange: Trial mode
  - 🔴 Red: Expired/Not installed

### 📥 **Download IDM**
- Direct link to official IDM website
- Quick access for fresh installation

### 🎨 **User Interface**
- Modern dark theme design

- Responsive button states
- Clear status indicators

### 🔧 **Technical Features**
- Automatic IDM process management

- File version detection
- Silent mode support (`/trial` command line)
- Single instance enforcement

## 🚀 How It Works

### **Freeze Trial Method**
1. Closes all IDM processes
2. Applies trial freeze patch
3. Restarts IDM automatically
4. Trial period remains at 30 days permanently

### **Reset Trial Method**
1. Closes all IDM processes
2. Cleans all IDM registry entries
3. Restarts IDM with fresh trial
4. Resets counter to 30 days

### **Activation Method**
1. Closes all IDM processes
2. Applies registration with custom name
3. Restarts IDM automatically
4. IDM becomes fully registered
